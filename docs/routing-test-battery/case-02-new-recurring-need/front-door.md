# Request - Front Door

## 1. What do you need?
A monthly partner health report that pulls together how each of our integration
partners is doing - ticket volume, escalations, and renewal risk - into a single
page our partnerships lead can scan before the monthly partner review.

## 2. Have we done this before?
No, this is new. We have been assembling it by hand in a doc each month and it is
painful.

## 3. One-time or recurring?
Recurring - every month, ahead of the partner review meeting.

## 4. By when?
The first one would be good before the next review on June 15, but I care more
about setting it up properly than rushing the first cut.

## 5. How sensitive is the data?
Internal. Some partner names and ticket counts, nothing regulated.

# Request — One-Off Assist (Funnel D)

Use this for a single, non-recurring task — something valuable to do once that
doesn't need a standing workflow. The operator does it directly; no workflow is
created. If this turns out to be a recurring need, it becomes a new-workflow intake
instead (see `intake-template.md`).

## 1. What needs to be done?
Describe the one-time task and what "done" looks like.

## 2. Input material
Attach or paste everything needed to do it.

## 3. Audience
Who is the finished output for?

## 4. Desired format
What should the deliverable look like (format, length, tone)?

## 5. Deadline / constraints
When do you need it, and any other constraints (tooling, sensitivity, length).
